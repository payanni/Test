# ALAN Immo

Een statische Nederlandstalige vastgoedsite geinspireerd op de look and feel van ALAN: zachte blauwtinten, veel witruimte, afgeronde kaarten en Alan Sans als primaire font.

## Functionaliteit

- Vastgoedcatalogus met filters voor locatie, type, budget, slaapkamers, oppervlakte, EPC-score en buitenruimte.
- Verkopersdashboard met bezoekersinzichten per pand.
- Bezoekmomenten beheren en bezoekers inschrijven.
- Leads scoren en automatisch rangschikken op potentieel.

## Lokaal draaien

```bash
python3 -m http.server 4173
```

Open daarna `http://localhost:4173`.
